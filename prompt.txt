У меня есть несколько кластеров с элементами, и для каждого из них требуется название, которое кратко(максимум 7-8 слов) и точно передает суть содержимого кластера. Используй названия элементов, чтобы придумать наиболее подходящее название для каждого кластера. Верни результат в формате JSON с полями:

"number" — номер кластера
"name" — сгенерированное название кластера.
Формат ввода:
Я даю тебе несколько кластеров.
Каждый кластер представлен списком элементов и имеет номер.
Формат вывода:
Верни ответ в виде JSON массива, где каждый объект представляет один кластер с номером и сгенерированным названием:

"number": номер кластера
"name": сгенерированное название кластера на основе его содержимого
Пример:
Ввод:
Кластер 1_2_3:
- АСУ «Подрядчик»
- Спецификации
- Протоколы
- Акты по электроэнергии
- Переход В Личный Кабинет Организации
- Контактная карта и схемы проезда
- Приложение Г. ФОРМА заявки на обеспечение СИЗ посетителей

Кластер 5_3_7:
- Отчеты о расходах
- Документы по инвентаризации
- Финансовые ведомости
- Учет доходов и расходов
- Планирование бюджета
Ожидаемый вывод(в формате json):
[
    {
        "number": "1_2_3",
        "name": "Документы и приложения для управления подрядчиками и электроэнергией"
    },
    {
        "number": "5_3_7",
        "name": "Финансовая отчетность и бюджетное планирование"
    }
]
Дополнительные требования:
Используй слова и темы из списка элементов для создания названия.
Если есть слова нежелательного характера, игнорируй их
Название должно быть кратким, но емким и передавать основную идею кластера.
